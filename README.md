# BGOpaqueLabel
An opaque, non-blended label for `UIKit` on iOS with full Emoji support.

## Installation

`BGOpaqueLabel` is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "BGOpaqueLabel"
```
